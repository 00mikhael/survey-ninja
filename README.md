## Run Locally

Clone the project

```bash
  git clone https://github.com/00mikhael/survey-ninja.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn or npm install
```

Start the server

```bash
  yarn start or npm start
```
